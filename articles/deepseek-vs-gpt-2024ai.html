    核心观点二：代码生成质量实测

    实战测试：生成一个复杂的数据处理函数

    我们让两个模型生成一个处理CSV数据并计算统计信息的函数，要求包含异常处理、类型推断和结果格式化。 DeepSeek输出： 
    import csv
import statistics
from typing import List, Dict, Union, Optional

def analyze_csv(file_path: str, columns: Optional[List[str]] = None) -> Dict[str, Dict[str, Union[float, str]]]:
    """
    分析CSV文件的数值列，返回基本统计信息
    
    Args:
        file_path: CSV文件路径
        columns: 要分析的列名列表，默认分析所有数值列
    
    Returns:
        包含每列统计信息的字典
    """
    result = {}
    
    try:
        with open(file_path, 'r', encoding='utf-8') as f:
            reader = csv.DictReader(f)
            data = list(reader)
            
            if not data:
                return {"error": "Empty CSV file"}
            
            # 自动检测数值列
            numeric_columns = []
            for col in data[0].keys():
                try:
                    float(data[0][col])
                    numeric_columns.append(col)
                except (ValueError, TypeError):
                    continue
            
            target_columns = columns if columns else numeric_columns
            
            for col in target_columns:
                values = []
                for row in data:
                    try:
                        values.append(float(row[col]))
                    except (ValueError, KeyError, TypeError):
                        continue
                
                if values:
                    result[col] = {
                        "count": len(values),
                        "mean": round(statistics.mean(values), 2),
                        "median": round(statistics.median(values), 2),
                        "stdev": round(statistics.stdev(values), 2) if len(values) > 1 else 0,
                        "min": round(min(values), 2),
                        "max": round(max(values), 2)
                    }
    
    except FileNotFoundError:
        return {"error": f"File {file_path} not found"}
    except Exception as e:
        return {"error": str(e)}
    
    return result
    GPT-4o输出（类似，但多了pandas版本）：

import numpy as np

def analyze_csv_pandas(file_path: str) -> dict:
    """使用pandas分析CSV"""
    df = pd.read_csv(file_path)
    numeric_cols = df.select_dtypes(include=[np.number]).columns
    
    stats = {}
    for col in numeric_cols:
        stats[col] = {
            'mean': df[col].mean(),
            'median': df[col].median(),
            'std': df[col].std(),
            'min': df[col].min(),
            'max': df[col].max(),
            'missing': df[col].isna().sum()
        }
    return stats
